Descripción detallada

Clase Abstracta Solution.

En esta clase, se define la estructura general de las distintas soluciones a ser codificadas.

Al ser una clase general, los métodos más particulares y específicos que se requieran para cada una de las distintas codificaciones, se definen como virtuales. En todas las subclases de Solution, obligatoriamente se deben redefinir esos métodos. Los cuales son:

  • getValues
  • Distance
  • ExtendSolution
  • show_solution

De los cuales se comenta más abajo.
